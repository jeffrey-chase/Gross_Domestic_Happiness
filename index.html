The dataset used to produce this visualization is in the form of a table with 155 rows each representing a different country (primary item type) and six different columns each representing a unique ordinal attribute served quantitatively by numbers representing the attribute's value for the corresponding item

These numbers are an estimate magnitude of six factors:

1. Gross Domestic Product

2. Social Support

3. Expectancy

4. Freedom

5. Generosity

6. Gov. Corruption

When combined these attributes have produced a list of the happiest countries in the world.
